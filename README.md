# battleship-game

Player vs. AI Battleship game created using HTML&CSS and vanilla JavaScript 
<br>
Live demo: https://jetskiis.github.io/battleship-game/
<br>
Uses unit testing via Jest and module bundling using Webpack
<br>
<br>
Features:
<br>
Users can input their names and place their own battleships
<br>
Smart AI that locks onto adjacent cells after a ship is hit.
